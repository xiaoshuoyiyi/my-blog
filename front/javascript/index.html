# 前言

JavaScript 是一种非常松散的面向对象语言，也是 Web 开发中极受欢迎的一门语言。JavaScript，尽 管它的语法和编程风格与 Java 都很相似，但它却不是 Java 的“轻量级”版本，甚至与 Java 没有任何关 系。JavaScript 是一种全新的动态语言，它植根于全球数亿网民都在使用的 Web 浏览器之中，致力于增 强网站和 Web 应用程序的交互性。

JavaScript 和 ECMAScript 通常都被人们用来表达相同的含义。

说的通俗点，这个可是前端吃饭的家伙，不能丢！

# 数据类型

JavaScript 中有 5 种简单数据类型（也称为基本数据类型）：Undefined、Null、Boolean、Number 和 String。还有 1种复杂数据类型——Object，Object 本质上是由一组无序的名值对组成的。

//基本类型
typeof(undefined);//undefined
typeof('123');//string
typeof(100);//number
typeof(true);//boolean

//引用类型
typeof(null);//object
typeof({});//object
typeof([]);//object
typeof(function (){});//function
